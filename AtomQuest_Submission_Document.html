Implemented Scope

  • Employee goal creation, validation, and submission
  • Manager inline review, approval, return for rework, and goal locking
  • Admin cycle management, exception unlock, and org hierarchy management
  • Shared departmental KPIs and linked achievement sync
  • Quarterly achievement capture and manager check-in comments
  • Progress scoring for Min, Max, Timeline, and Zero-based goals
  • CSV achievement report, completion dashboard, and audit trail
  • Configurable escalation rules, escalation log, and analytics dashboard
  • Microsoft Entra/email/Teams configuration readiness with simulated notifications

Suggested Demo Flow

  1. Start the app with npm start and open http://127.0.0.1:3000.
  2. Login as Admin / HR and open Phase 1 goal-setting window.
  3. Login as Employee and create/submit goals.
  4. Login as Manager and approve or return submitted goals.
  5. Login as Admin / HR and open a quarterly check-in window.
  6. Login as Employee and enter achievements.
  7. Login as Manager and record check-in comments.
  8. Download the CSV report as Admin / HR or Manager.

Real Microsoft Entra SSO, email delivery, and Teams bot delivery require external tenant/webhook credentials. This project includes config-ready settings and simulated in-app notifications for hackathon demonstration.

# AtomQuest Hackathon 1.0 - Goal Setting & Tracking Portal

A complete browser-based Goal Setting and Tracking Portal built for the AtomQuest Hackathon problem statement.

## Tech Stack

- Backend: Node.js built-in HTTP server, no external packages
- Frontend: Vanilla HTML, CSS, and JavaScript SPA
- Database: JSON file persistence at `backend/data/db.json`
- Reports: CSV export generated by backend

This keeps the demo low-cost and easy to host on a small VM, Render, Railway, Azure App Service, or any Node-compatible platform.

## Run Locally

```bash
npm start
```

Open `http://localhost:3000`.

## Implemented Requirements

- Employee goal sheet creation and submission
- Thrust Area, title, description, UoM, target, deadline, and weightage fields
- Validation for total weightage = 100%, minimum 10%, and maximum 8 goals
- L1 manager review, inline edits, approve-and-lock, and return-for-rework
- Locked goals cannot be edited except through Admin exception handling
- Shared departmental KPIs pushed by Admin or Manager
- Shared KPI title and target are read-only for recipients; weightage remains editable
- Achievement updates by a shared goal primary owner sync to linked goal sheets
- Quarterly achievement capture for Q1, Q2, Q3, and Q4 windows
- Goal status updates: Not Started, On Track, Completed
- Manager check-in comments per employee and quarter
- Computed progress formulas for Min, Max, Timeline, and Zero-based goals
- Admin cycle/window management matching the required quarterly schedule
- CSV achievement report
- Completion dashboard for employee and manager check-ins
- Audit trail for workflow events, locked-goal changes, achievement updates, cycle changes, and unlocks
- Admin org hierarchy and goal unlock capability

## Bonus Features Included

- Simulated in-app notifications for email/Teams-style event alerts
- Rule-based escalation log
- Analytics dashboard with completion heatmap, goal distribution, status mix, and manager effectiveness
- Cost-optimized architecture with no paid dependencies required for demo

## Suggested Demo Flow

1. Login as Employee and add draft goals until weightage totals 100%.
2. Submit the goal sheet.
3. Login as Manager, open Approvals, adjust a target or weightage inline, then approve and lock.
4. Login as Admin, open the Q1/Q2/Q3/Q4 check-in window from Admin / HR.
5. Login as Employee and enter actual achievements.
6. Login as Manager and record a structured check-in comment.
7. Login as Admin or Manager and download the CSV report.

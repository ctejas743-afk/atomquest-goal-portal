Summary:
This is a complete web-based Goal Setting & Tracking Portal implementing Employee, Manager, and Admin journeys. It supports goal creation, validation, submission, manager approval, locked goals, admin unlock, shared KPIs, quarterly achievements, check-ins, progress scoring, CSV reporting, completion dashboard, audit logs, hierarchy management, escalation rules, analytics, and integration-readiness settings.
